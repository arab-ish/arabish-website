# Arab-ish Website

Production-ready starter website for Arab-ish Mediterranean Fusion.

## What is included

- Next.js app router website
- Tailwind CSS styling
- Official Arab-ish logo in `/public/arabish-logo.png`
- Mobile-first homepage
- Catering inquiry form layout
- SEO metadata for Madison WI food truck and catering keywords
- Static export configuration for Cloudflare Pages

## Cloudflare Pages settings

When connecting this repository to Cloudflare Pages, use:

- Framework preset: Next.js
- Build command: `npm run build`
- Build output directory: `out`

## Important edits before final launch

Replace placeholder content:

- Schedule
- Phone number
- Email address, if needed
- Instagram handle
- Real food/trailer photos
- Map embed or live location link
- Form backend integration

The logo file must not be modified.
